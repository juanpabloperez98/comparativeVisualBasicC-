Estructuras de selección simples

¿Que son las estructuras de selección simple?

Las estructuras de selección simples permiten que el flujo del programa siga por un camino especifico si se cumple una condición o conjunto de condiciones. Las estructuras de selección simple se utilizan utilizando una sentencia llamada If, esta sentencia condicional se usa para tomar decisiones, este evalúa básicamente una operación lógica, es decir una expresión booleana de resultado True o False y permite la ejecución de un fragmento de código dado el caso de la condición lógica se cumpla o no.

Sintaxis

Teniendo un poco de idea de lo que es un condicional if ahora se explicará sobre su sintaxis. La sintaxis de la sentencia if varia un poco en el lenguaje Visual Basic con respecto al lenguaje C#, a pesar de esto la idea lógica es la misma. A continuación, un ejemplo de como esta compuesta la sintaxis en cada uno de estos lenguajes.

Sintaxis C#
