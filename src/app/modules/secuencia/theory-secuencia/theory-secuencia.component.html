Tipos de datos

¿Que son los tipos de datos?

En programación, un tipo de dato informático o simplemente tipo es un atributo de los datos que indica al ordenador sobre la clase de datos que se va a trabajar. Esto incluye imponer restricciones en los datos, como qué valores pueden tomar y qué operaciones se pueden realizar. Un tipo de dato es un espacio en memoria con restricciones. Por ejemplo, el tipo "int" representa, generalmente, un conjunto de enteros de 32 bits cuyo rango va desde el -2.147.483.648 al 2.147.483.647, así como las operaciones que se pueden realizar con los enteros, como son la suma, la resta, y la multiplicación.

Variables y constantes

Una variable es un nombre para identificar una posición en memoria. En estas se guardan los datos usados por el programa durante su ejecución. Las variables deben ser declaradas antes de poder ser usadas. Las constantes son valores que no son cambiados una vez el programa compile.

Ver ejemplo 
          {{ CODECSHARP1 }}
          
        
          {{ CODECVB1 }}

Resultado

          
            10
            69

Para definir una variable o una constante en C # como en Visual Basic , es necesario declarar que tipo de dato con el que se va a manejar, más abajo se encontraran los conceptos de tipos de variables que manejan tanto C # como Visual Basic, para este caso se manejaron los tipos de datos Integer o Int (Son lo mismo)

Identificadores

Siempre que se vayan a declarar variable o constantes es necesario poder crearlas con un identificador, para poder utilizar esta variables o constantes posteriormente en algún punto del código. Los identificadores no son más que el nombre que se le da a una variable o a una constante cuando son declaradas. Estos identificadores cuentan con algunas restricciones:

  • Debe iniciar con un carácter alfabético o el simbolo: "_".
  • No debe tener caracteres especiales como: ( @ # / “ & ).
  • No debe haber espacios en blanco.
  • Se tienen en cuenta mayúsculas y minúsculas.
  • Las palabras reservadas no pueden ser usadas como identificadores.
Ver ejemplo 
            {{ CODECSHARP2 }}
            
          
            {{ CODECVB2 }}
