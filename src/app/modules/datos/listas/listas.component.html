Estructuras de datos

Arrays, arreglos o vectores

¿Que son los arrays, arreglos o vectores?

Estos son el tipo de estructura de dato más conocida en programación, y no es de extraño ya que la utilidad se basa en ser utilizados para almacenar múltiples valores en una única variable. En términos de arquitectura de computadores, estos apartan espacio o posiciones en memoria continuas, permitiendo que se pueda acceder a ellos rápidamente, se usan cuando se necesita guardar más de un dato o información en una variable. La analogía de los vectores puede tomarse como un tren de N vagones, donde cada vagón es un espacio de memoria que almacena una variable.

Sintaxis

La sintaxis de las listas o arreglos suelen ser teóricamente muy similares tanto en VB como en C#, y se dice que teóricamente puesto que el concepto para poder definir un arreglo es el mismo, pero la sintaxis o la manera de escribirlos o declararlos cambian un poco dependiendo el lenguaje, a continuación, se mostrara acerca de la sintaxis de los arreglos en los dos lenguajes de programación que estamos trabajando.

Sintaxis VB

En Visual Basic la declaración de un array, arreglo o vector se empieza utilizando la palabra reservada Dim definiendo posteriormente el nombre de la variable que va almacenar el arreglo y el tipo de dato, es decir (String, Integer), es importante recalcar que los arreglos no pueden ser infinitos

Sintaxis C#

Para declarar un arreglo en C# se deben colocar los corchetes después de tipo de dato. Es decir, se debe declarar el tipo de dato del arreglo, seguido de los corchetes y luego el nombre del array. Esta forma de declarar un array se utiliza cuando se desea declarar un array sin inicializar, es decir sin definir el tamaño del array. La otra forma de declarar un array es definiendo el tamaño del mismo, para esto solo se va a declarar una variable y se iguala a la expresión new tipo_de_dato[10], es de esta manera como se guardara en la variable definida el arreglo creado
