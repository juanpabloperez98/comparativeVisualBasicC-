Estructura repetitiva WHILE

¿Que son las Estructura repetitiva WHILE?

La estructura while repite un bloque de instrucciones mientras una condición dada se cumpla, es decir, sea verdadera, ¿qué quiere decir esto? el ciclo while puede repetirse un sin fin de veces, lo que significa que no es necesario definir el número de veces que se va a repetir el ciclo. En los ciclos while no hay necesidad de tener un valor inicial ni un valor fina. Muy similar a las estructuras de selección, esta estructura repetitiva depende de una condición la cual se evalúa antes de ejecutar el cuerpo de bucle, por lo que, si la condición no se cumple, ninguna instrucción dentro de la estructura se ejecutará.

Sintaxis

La sintaxis de un ciclo while, es de hecho más simple que la de un ciclo for. Solo es necesario tener claro la condición de finalización. Normalmente para todos los lenguajes de programación se maneja la misma estructura en este ciclo, y no es la excepción para los lenguajes tanto como para VB como para C#

Sintaxis VB

Esta sintaxis es definida por la palabra reservada While acompañada de una expresión booleana, es decir una expresión que retorna verdadero o falso, dentro de esta estructura, se ejecuta un número indefinido de veces las sentencias de bloques que se encuentre en ella. 

            {{ CODECSHARP1 }}

Sintaxis C#

La sintaxis para este lenguaje es similar a la de VB, viene acompañada de la palabra reservada while acompañada entre paréntesis de una expresión booleana, la diferencia mas significativa son las llaves que se deben utilizar para ejecutar el bloque de código que hayan dentro de ellas 

            {{ CODECVB1 }}
