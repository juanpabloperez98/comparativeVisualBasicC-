Aprendizaje de programación imperativa en visual Basic y C Sharp

Este software es desarrollado por la universidad tecnológica de Pereira, en este se busca tener una introducción y un proceso de aprendizaje de la programación imperativa en los lenguajes de programación Visual Basic y C Sharp, adicional a esto se tiene como objetivo servir de herramienta de apoyo para los estudiantes que apenas se están adentrando al mundo de la programación imperativa, la idea principal del software es poder explicar la programación imperativa en los lenguajes mencionados anteriormente haciendo una comparativa sobre como realizar diversos problemas que se pueden encontrar a la hora de programar.

¿De que trata este software?

Como se menciono anteriormente, el software busca ser una guía o herramienta de apoyo para las personas que apenas se estén adentrando en el mundo de la programación imperativa, haciendo una comparación entre los lenguajes Visual Basic y C Sharp, es por eso que este software tiene un apartado total de 6 módulos, en los cuales se podrán encontrar temas en relación a cada módulo, cada tema o sección contara con unos ejemplos prácticos y unos ejercicios con los cuales se busca afianzar el conocimiento adquirido en dichas secciones. El estudiante es libre de escoger el módulo al que desea entrar, se recomienda llevar la secuencia de los módulos para mayor entendimiento y compresión

¿Por que C# y Visual Basic?

Antes de empezar hablar de Visual Basic o C Sharp es importante mencionar o hablar de lo que es .NET, básicamente .NET es una plataforma que nos proporciona Microsoft la cual viene integrada con diferentes herramientas que permiten el desarrollo de aplicaciones, de esas herramientas mencionadas están incluidas los lenguajes de programación Visual Basic y C Sharp. Los dos lenguajes de programación soportan paradigma imperativo, es por eso que el desarrollo de esta guía es basado en estos lenguajes, debido a su similitud y componentes que comparten.

Teoría

Cada módulo está compuesto de secciones, las cuales busca enseñar temas puntuales y específicos que se manejan en la programación imperativa, el conjunto de estas secciones conforma un módulo, cada sección tiene una parte teórica la cual busca adentrar al estudiante sobre los temas propuestos.

Practica

Así mismo cada sección que hace parte de un módulo contiene su parte práctica, este apartado tiene como fin, poder probar la habilidad y la destreza del estudiante que ha adquirido en la sección

Puntos a tocar en este software

Con el objetivo de poder cumplir con los criterios y formatos de evaluación, se menciona a continuación cuales objetivos son los que este software abarca:

  • ROBUSTEZ: Este software cuenta con un manual de usuario y un manual técnico el cual le permite tanto a la persona que lo va a usar como el que quiere saber como esta desarrollado el mismo tener una visión clara sobre el software y sus secciones más significativas
  • EXTENDIBILIDAD: A pesar de que este software no cuenta con una facilidad de añadírseles nuevas características, se cuenta con la documentación adecuada para poder implementar nuevas propiedades
  • MANTENIMIENTO: Se presenta una buena documentación para realizar cambios en el mismo
  • GRADO DE INVENCIÓN O INNOVACIÓN: A pesar de que SML es un lenguaje conocido, es difícil encontrar tutoriales y ejemplos de este lenguaje de programación, adicional de eso, este software cuenta con un sistema de ejemplos los cuales muestran el paso a paso de diferentes ejercicios aplicados en el lenguaje
